## 自我介紹 (Summary)

我對技術的執著源自對「系統如何運作」的本質好奇。從國中自學遊戲開發、高中完成跨國接案，到現在獨立經營 **Mindoyo**——一個以我為核心的創作與工程工作室，負責從問題定義到部署上線的完整產品週期。

目前正在部署 **Agora-AI**：一個 LLM 驅動的學生行政助理。學校的 API 沒有任何文件，所以我把它逆向出來，包成 10 個 agent 工具，讓學生用自然語言查詢校務。同步進行的還有校園點餐系統，以及授權範圍內持續進行的資安評估與修補工作。
